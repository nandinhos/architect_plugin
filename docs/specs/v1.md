# 🏗️ Architect Engine — Especificação Técnica (v1.0)

## 1. Objetivo

Formalizar o funcionamento do **Behavior Engine** como um sistema executável, determinístico e extensível, responsável por impor padrões de engenharia sobre saídas de IA.

---

## 2. Definição do Core

O Architect Engine é um **runtime de regras comportamentais** que atua em pipelines de geração de código.

### 2.1 Conceito

- Entrada: contexto + output de IA
- Processamento: regras (validação + enforcement)
- Saída: código validado, corrigido ou bloqueado

---

## 3. Arquitetura Geral

```text
[AI Output]
     ↓
[Context Builder]
     ↓
[Behavior Engine]
     ↓
 ├── Rule Engine
 ├── Security Layer
 ├── Design Validator
 └── Test Enforcer
     ↓
[Validated Output]
```

---

## 4. Contrato de Regras

### 4.1 Interface Base

```ts
export interface BehaviorRule {
  id: string;
  name: string;
  trigger: TriggerType;
  severity: 'low' | 'medium' | 'high' | 'critical';
  validate(context: RuleContext): RuleResult;
  enforce?(context: RuleContext): EnforcementResult;
}
```

### 4.2 Tipos

```ts
type TriggerType = 'before_generation' | 'after_generation' | 'pre_commit' | 'review';

interface RuleContext {
  code: string;
  filePath: string;
  metadata: Record<string, any>;
}

interface RuleResult {
  valid: boolean;
  issues: Issue[];
}

interface Issue {
  message: string;
  line?: number;
  severity: string;
}
```

## 5. Pipeline de Execução

### 5.1 Fluxo

1. Receber output da IA
2. Construir contexto
3. Executar regras por trigger
4. Agregar resultados
5. Decidir ação

### 5.2 Decisão

```text
Se houver CRITICAL → BLOQUEIA
Se houver HIGH → exige correção
Se houver MEDIUM → alerta
Se LOW → log
```

## 7. Registro de Regras

```ts
class RuleRegistry {
  private rules: BehaviorRule[] = [];

  register(rule: BehaviorRule) {
    this.rules.push(rule);
  }

  getByTrigger(trigger: TriggerType) {
    return this.rules.filter((r) => r.trigger === trigger);
  }
}
```

---

## 8. Engine Principal

```ts
class ArchitectEngine {
  constructor(private registry: RuleRegistry) {}

  run(context: RuleContext, trigger: TriggerType) {
    const rules = this.registry.getByTrigger(trigger);

    const results = rules.map((rule) => ({
      rule: rule.id,
      result: rule.validate(context),
    }));

    return this.evaluate(results);
  }

  private evaluate(results: any[]) {
    const issues = results.flatMap((r) => r.result.issues);

    if (issues.some((i) => i.severity === 'critical')) {
      return { status: 'blocked', issues };
    }

    return { status: 'ok', issues };
  }
}
```

## 10. Exemplo de Regra

```ts
const noRawSQLRule: BehaviorRule = {
  id: 'SEC-001',
  name: 'No Raw SQL Injection Risk',
  trigger: 'after_generation',
  severity: 'critical',

  validate(context) {
    const hasRisk = context.code.includes('SELECT * FROM');

    if (hasRisk) {
      return {
        valid: false,
        issues: [
          {
            message: 'Possível SQL Injection detectado',
            severity: 'critical',
          },
        ],
      };
    }

    return { valid: true, issues: [] };
  },
};
```
