# AGENTS.md — The Architect Protocol Interface

Este projeto opera sob o protocolo de engenharia do **Arquiteto Sinistro**.
Qualquer agente de IA (Claude, Gemini, GPT-4o, Cursor) deve aderir aos seguintes contratos:

1. **Ponto de Entrada:** Comece sempre lendo o arquivo `ARCHITECT.md`.
2. **DNA Visual:** Toda UI deve ser baseada nos tokens em `.architect/design/tokens.json`.
3. **Comportamento Sênior:** Siga os protocolos de TDD, Segurança e Ignorância Técnica em `.architect/skills/senior-engineer.md`.
4. **Manifestos:** Consulte o diretório `.architect/manifests/` para instruções específicas para sua ferramenta.
